# 🚀 OpportunityPilot AI

An AI-powered career guidance platform that helps students discover internships, hackathons, scholarships, eligibility, and personalized learning roadmaps using Google's Gemini AI.

---

## ✨ Features

- 🔍 Opportunity Scout
- ✅ Eligibility Analyzer
- 📈 3-Month Career Roadmap
- 🤖 Powered by Gemini AI
- 🎨 Modern Streamlit Dashboard
- 📥 Download Reports
- 🌙 Responsive Dark UI

---

## 🛠 Tech Stack

- Python
- Streamlit
- Google Gemini API
- HTML/CSS
- Markdown

## 🚀 Installation

Clone the repository

```bash
git clone <repository-link>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
streamlit run app.py
```
